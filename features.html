The Loruki Platform

Spin up Droplet cloud servers with industry leading price-performance and predictable costs.

Droplets are flexible Linux-based virtual machines that run on top of virtualized hardware. Each Droplet you create is a new cloud server you can use.

Simple or bursty applications such as low traffic web servers, blogs, discussion forums, CMS, small databases, dev/test servers, microservices, and repository hosting.

Run your apps on modern hardware that delivers the fast performance your customers demand.

The most basic Droplet – a burstable portion of vCPU – along with a configurable amount of memory.

Premium Droplets are ideal for in-memory databases and server-side caches for web apps, workloads that require a large number of transactions.

Intel Premium Droplets have second generation Xeon Scalable processors (2.5 GHz, 3.9 GHz turbo and feature memory frequency of 2933 MHz.
